# Data Reconciliation & Insights Tool

## Project Overview

I built this project to simulate a real business challenge where data from different systems does not match.

The tool compares records from an API source and a database source to identify issues such as missing records, incorrect values, and mismatched totals. Instead of only showing that there is a problem, it also helps explain why the mismatch happened.

This project helped me strengthen my Python, SQL, and debugging skills while working on something that reflects real-world backend and data investigation tasks.

---

## Features

- Compare data between an API source and a database
- Extract and validate records using SQL queries
- Detect missing records and incorrect values
- Identify the reason behind each mismatch
- Display results in a simple Streamlit dashboard

---

## Tech Stack

- Python
- SQL (SQLite)
- Pandas
- Streamlit
- JSON
- Visual Studio Code

---

## Example Mismatches Detected

Some of the issues the tool can detect include:

- Records missing from the API
- Records missing from the database
- Incorrect transaction amounts
- Total value mismatches between systems
